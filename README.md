# Smart Campus Management Integrated Platform

A full-stack web application developed for managing campus activities digitally.  
This project provides separate dashboards for Students, Faculty, and Admin with role-based access.

## Features

### Student Module
- User Registration
- Login Authentication
- Attendance Management
- Fee Management
- Academic Performance
- Hostel Information
- Notifications
- Student Profile
- Analytics Charts

### Faculty Module
- Attendance Management
- Upload Marks
- Class Schedule
- Assignments
- Notifications
- Student Reports

### Admin Module
- Manage Students
- Manage Faculty
- Fee Management
- Hostel Management
- Notifications
- Reports
